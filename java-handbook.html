01Foundations ▸ What is Java

      
What is Java?

      
Java is a statically typed, object-oriented language that compiles to platform-neutral bytecode. That bytecode runs on the Java Virtual Machine (JVM), which is why the same compiled program runs on Windows, macOS, and Linux without changes — the famous “write once, run anywhere.”

      
    
        
  • JDK — Java Development Kit: the compiler (javac), tools, and the JRE. You install this to write Java.
    • 
        
  • JRE — Java Runtime Environment: the JVM plus core libraries needed to run programs.
    • 
        
  • JVM — the engine that loads, verifies, and executes bytecode, and manages memory via garbage collection.
    • 
      

      
The flow: .java source → javac compiles → .class bytecode → the JVM runs it.
